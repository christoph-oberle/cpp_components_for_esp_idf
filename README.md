# C++ Components for ESP-IDF

This repository contains an example how to use 
[elrebo-de/wifi_manager](https://components.espressif.com/components/elrebo-de/wifi_manager/versions/1.4.2/readme) and 
[elrebo-de/deep_sleep](https://components.espressif.com/components/elrebo-de/deep_sleep/versions/1.2.1/readme) 
in your own ESP32 programs.

It shows how to start with an empty program, add Wi-Fi capabilities and the capability to enter Deep Sleep mode in only six steps and 
with only 75 lines of code.

A description of this example can be found on 
[the Espressif Developer Blog](https://developer.espressif.com/blog/2026/04/cpp-components-for-esp-idf/) (publishing date: 13.04.2026).
